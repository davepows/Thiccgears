# ThiccGears

A parody competing website for [ThinGears](https://thingears.com).

If ThinGears makes impossibly thin aluminum gears for FRC robots, ThiccGears makes the opposite: the beefiest, most structurally intimidating gears on the market. Zero apologies.

## Stack

Single-file HTML/CSS site. No dependencies, no build step. Just open `index.html`.

## Products

- **ThiccGear Chonk** — 12mm, entry level beef
- **ThiccGear Beefcake** — 22mm, the gold standard
- **ThiccGear Absolute Unit** — 31mm, ships in a reinforced crate

## Deploying

Works as a static site — drop on GitHub Pages, Netlify, Vercel, wherever.
